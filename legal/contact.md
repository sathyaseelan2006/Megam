## Bug Reports

Found a bug? Please report it on GitHub with:

1. **Description**: What happened?
2. **Steps to Reproduce**: How can we recreate the issue?
3. **Expected Behavior**: What should have happened?
4. **Screenshots**: Visual evidence (if applicable)
5. **Environment**: Browser, OS, device

## Security Issues

If you discover a security vulnerability, please **DO NOT** open a public issue. Instead:

1. Email us directly at [security-email@example.com] *(set up a security email)*
2. Include detailed steps to reproduce
3. Allow reasonable time for us to fix before public disclosure

We take security seriously and will respond promptly to all security reports.
